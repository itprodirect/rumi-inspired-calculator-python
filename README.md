# Rumi-Inspired Calculator (Python + Tkinter)

A small GUI calculator themed after Rumi from *K-pop Demon Hunters* — dark stage background, neon accents, and keyboard support.

## Setup

```bash
git clone https://github.com/itprodirect/rumi-inspired-calculator-python.git
cd mira-inspired-calculator-python
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
python rumi_calculator.py
```

## Features

* Basic arithmetic: +, -, ×, ÷
* Keyboard input (numbers, operators, Enter, Backspace, Esc)
* Rumi-themed neon UI using Tkinter
